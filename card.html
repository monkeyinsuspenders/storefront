How to add new safe fields later
1. Add a new form field inside the form.
2. Give it a unique id.
3. Add a matching preview element if you want it shown on the badge.
4. Add one JavaScript line in bindText() near the bottom.
Example: add an Office Location field
<div class="mt-3">
  <label class="form-label" for="officeLocation">Office Location</label>
  <input type="text" class="form-control" id="officeLocation">
</div>
Example preview block
<div class="badge-line">
  <div class="small-label">Office</div>
  <div id="previewOfficeLocation">Philadelphia</div>
</div>
Example JavaScript binding line
bindText("officeLocation", "previewOfficeLocation", "Philadelphia");
